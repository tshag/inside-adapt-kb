# Claims Submission Protocol

## Overview

This protocol outlines the standard procedures for submitting insurance claims at Adapt Psychiatry.

## Pre-Submission Checklist

Before submitting a claim, verify:

- [ ] Patient insurance information is current
- [ ] Authorization obtained (if required)
- [ ] Correct CPT codes selected
- [ ] ICD-10 diagnosis codes documented
- [ ] Provider NPI included
- [ ] Place of service code correct
- [ ] Dates of service accurate

## CPT Code Reference

### Common Psychiatry Codes

| Service | CPT Code | Description |
|---------|----------|-------------|
| Initial evaluation | 90792 | Psychiatric diagnostic evaluation with medical services |
| Follow-up 15 min | 99213 | Office visit, established patient |
| Follow-up 25 min | 99214 | Office visit, established patient |
| Follow-up 40 min | 99215 | Office visit, established patient |
| Therapy 45 min | 90834 | Psychotherapy, 45 minutes |
| Therapy 60 min | 90837 | Psychotherapy, 60 minutes |
| E/M + Therapy | 99213/99214 + 90833/90836 | Combined services |

## Submission Timeline

- **Clean claims:** Submit within 24 hours of service
- **Claims requiring auth:** Submit within 48 hours of receiving authorization
- **Corrected claims:** Submit within 7 days of identifying error

## Payer-Specific Requirements

### Commercial Insurance

1. Verify eligibility before each visit
2. Obtain prior auth for specific services
3. Submit electronically when possible
4. Follow up within 30 days

### Medicare

1. Ensure Medicare enrollment is current
2. Use appropriate modifiers
3. Submit within 1 year of service date
4. Monitor remittance advice weekly

### Medicaid

1. Verify Medicaid eligibility monthly
2. Obtain PA when required
3. Submit through eMedNY portal
4. Follow up within 45 days

## Denial Management

### Common Denial Reasons

| Denial Code | Reason | Resolution |
|-------------|--------|------------|
| CO-29 | Time limit expired | Submit with documentation |
| CO-45 | Charge exceeds fee schedule | Adjust per contract |
| CO-50 | Non-covered service | Verify benefits, appeal if needed |
| CO-252 | Missing/invalid info | Correct and resubmit |
| PR-2 | Deductible not met | Patient responsibility |

### Appeal Process

1. Review denial reason thoroughly
2. Gather supporting documentation
3. Submit appeal within payer deadline
4. Track appeal status
5. Escalate if denied again

## Key Performance Indicators

- **Clean claim rate:** Target >95%
- **Days in AR:** Target <35 days
- **Denial rate:** Target <5%
- **Collection rate:** Target >98%
