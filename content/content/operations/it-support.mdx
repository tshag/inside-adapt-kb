## Common Issues

### Password Reset

**For Google Workspace:**
1. Go to [Google Account Recovery](https://accounts.google.com/signin/recovery)
2. Enter your @adaptwny.com email
3. Follow verification steps
4. If unable to recover, contact IT

**For EHR System:**
1. Click "Forgot Password" on login page
2. Enter your username
3. Check email for reset link
4. Contact IT if no email received within 10 minutes

### VPN Connection

**Connecting to VPN:**
1. Open Cisco AnyConnect VPN client
2. Enter server: `vpn.adaptwny.com`
3. Enter your Adapt credentials
4. Click Connect

**Troubleshooting:**
- Ensure internet connection is stable
- Try different network (mobile hotspot)
- Restart VPN client
- Contact IT if still unable to connect

### Printer Issues

**Printer Not Responding:**
1. Check printer is powered on
2. Verify network connection (WiFi or Ethernet)
3. Restart printer
4. Check print queue for stuck jobs
5. Re-add printer if necessary

**Adding a Network Printer:**
1. Open Settings â†’ Printers & Scanners
2. Click "Add Printer"
3. Select printer from list
4. Install drivers if prompted
5. Contact IT for printer IP address if not found

### Email Issues

**Not Receiving Emails:**
1. Check spam/junk folder
2. Verify email address is correct
3. Check storage quota (Google Workspace)
4. Check email filters/rules

**Cannot Send Emails:**
1. Verify internet connection
2. Check recipient email address
3. Review attachment size (max 25MB)
4. Try sending from web interface

## Security

### Password Requirements

- Minimum 12 characters
- Uppercase and lowercase letters
- Numbers and special characters
- Changed every 90 days
- No reuse of last 5 passwords

### Two-Factor Authentication (2FA)

**Required for:**
- Google Workspace
- EHR system
- VPN access

**Setup:**
1. Use Google Authenticator app
2. Scan QR code during setup
3. Save backup codes securely
4. Contact IT if phone is lost/replaced

### Device Security

**Laptops/Workstations:**
- Lock screen when away (Windows+L)
- Never share login credentials
- Report lost/stolen devices immediately
- Keep operating system updated

**Mobile Devices:**
- Use device passcode/biometric
- Enable remote wipe capability
- Only install approved apps
- Keep work and personal data separate
